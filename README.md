# Examples from The SysML v2 Book

This repository contains example models from "The SysML v2 Book" from Tim Weilkiens and Vince Molnar published by [MBSE4U](https://mbse4u.com). You can buy the pre-release of the book on Leanpub: [https://leanpub.com/sysmlv2](https://leanpub.com/sysmlv2).

The book is still in progress, as are the examples.
